# ExamenRedes

Pasos para ejecutar (Servidor):
1. Primero hay que echar a correr el servidor, para eso hay que descargar apiExamen.py
2. Abrir una terminal en donde se encuentre el archivo descargado
3. Ejecutar el comando 'python3 apiExamen.py'
4. El server ya está corriendo

Pasos para ejecutar (Cliente):
1. Descargar el archivo clienteExamen.py
2. Abrirlo en un editor de código
